# **Paper A 草稿 (Draft v1.0)**

## **標題：**

**Beyond the Newtonian Limit: Globalizing Solvability in Highly Anisotropic Hamiltonian Constraints via Pseudo-Transient Continuation**
**(超越牛頓極限：通過偽瞬態延拓實現高各向異性 Hamiltonian 約束的全局可解性)**

**摘要 (Abstract):**
在數值相對論的初值問題中，為了處理無限邊界或黑洞視界，往往需要引入極端的坐標變換（如雙曲正弦映射）。這類變換雖然解決了幾何拓撲問題，卻引入了強各向異性（High Anisotropy, ），導致橢圓型約束方程的 Jacobian 矩陣條件數呈指數級惡化。本文報告了一個關鍵發現：當各向異性超過特定閾值時，標準的牛頓-克雷洛夫（Newton-Krylov）方法會遭遇「信任域崩塌（Trust Region Collapse）」，表現為殘差的停滯或數值振蕩。為此，我們提出了一種基於**偽瞬態延拓（Pseudo-Transient Continuation, PTC）**的全局化求解架構。通過引入虛擬時間步長  作為同倫參數，結合**自適應對角預條件**與**幀內重試（In-Frame Retry）**機制，我們成功恢復了求解路徑的嚴格單調性。數值實驗表明，該方法能夠在極端拉伸網格上穩健地收斂，為高精度黑洞模擬提供了一個最小可行的數值框架。

> **2.4 求解器的層級架構 (A Hierarchical Solver Suite)**
> 雖然本文重點探討極端各向異性下的廣義相對論（GR）求解，但我們的 KOUN-GR 架構實際上提供了一套針對不同物理情境的「分層解決方案」。並非所有問題都需要動用偽瞬態延拓（PTC）這種重型武器。為了在精度與計算成本之間取得最佳平衡，我們開發了以下三個層級的求解器變體：
> 1. **Level 0：固定網格牛頓求解器（Fixed-Grid Newtonian Solver）。**
> * **適用場景：** 弱引力場、無極端邊界條件的經典泊松方程（Poisson Equation）。
> * **特點：** 極致高效。利用標準的多重網格（MG）或 FFT 加速，可在毫秒級完成大規模計算。
> 
> 
> 2. **Level 1：變網格牛頓求解器（Variable-Grid Newtonian Solver）。**
> * **適用場景：** 需要處理無限邊界或特殊拓撲，但物理場本身仍處於線性或弱非線性區間（非相對論極限）。
> * **特點：** 結合了 Koun-U 坐標變換的幾何優勢，但保留了經典牛頓法的速度。
> 
> 
> 3. **Level 2：偽瞬態相對論求解器（PTC-GR Solver，即本文核心）。**
> * **適用場景：** 強引力場、黑洞視界附近、以及  的高各向異性區域。
> * **特點：** 犧牲部分速度以換取絕對的穩健性（Robustness）。它是唯一能在結構性斷裂區域維持可解性的工具。
> 
> 
> 
> 
> 這種分層設計確保了該架構在處理從「教科書級練習」到「前沿黑洞模擬」的全光譜問題時，始終能保持最優的效能比。
